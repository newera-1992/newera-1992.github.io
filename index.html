快捷键配置

JS格式化是与 SQL冲突、CSS格式化需要选择多项，所以修改默认值

路径：Perfercense > Key Bindings

粘贴到右边的用户设置框

[
    {
        "keys": ["ctrl+alt+j"], "command": "js_format",
        "context": [{"key": "selector", "operator": "equal", "operand": "source.js,source.json"}]
    },
    {
        "keys": ["ctrl+alt+["],
        "command": "css_format",
        "args": {
            "action": "expand"
        }
    },
    {
        "keys": ["ctrl+alt+]"],
        "command": "css_format",
        "args": {
            "action": "compress"
        }
    }
]

插件用法

  • MarkdownPreview

用法：ctrl+shift+p, 输入：Markdown Preview, 选择 save to html 或者 preview in browser

  • CSS Format

用法：ctrl+alt+[（展开）、ctrl+alt+]（压缩成一行）

  • ExportHTML

用法：ctrl+shift+p, 输入：export, 选择 show export menu

  • GenerateUUID

用法：ctrl+shift+u

  • JsFormat

由于与 sql格式华那个插件按键冲突，将其改成：ctrl+alt+j，配置代码在后面快捷键配置部分

js大文件可能会延迟一下才看到效果

用法：ctrl+alt+j

  • Indent XML

注：如果正常能搜索到的就直接从 ctrl+shift+p 安装就好了

安装：发现索引里面没有了，需要手动去 github.com/alek-sys/sublimetext_indentxml

一、解压包含目录到：sublime\Data\Packages

二、用法：ctrl+k,f

  • Random Text

默认大小字母数字字符32位，可设置

用法：ctrl+shift+r

  • Timenow

用法：ctrl+t

  • uroboroSQL Formatter

用法：ctrl+alt+f

  • ConvertToUTF8

用法：ctrl+shift+c

注：大文件可能会延迟一下才看到效果

  • YamlToJson

用法：ctrl+y, ctrl+j = yaml转json 用法：ctrl+j, ctrl+y = json转yaml

备份打包

便携版打包整个解压目录即可

注：如果是安装版, %UserProfile% 路径下是设置文件, 和软件目录是分离的。
