Tab字符 空格

历史记录 关闭
自动打开历史文件 关闭
自动保存 关闭

快捷键配置

路径：Perfercense > Key Bindings

名称 快捷键
JS/CSS格式化_展开 ctrl+alt+[
JS/CSS格式化_压缩成一行 ctrl+alt+]
JSON格式化 ctrl+k, ctrl+f
XML格式化 ctrl+k, ctrl+f
SQL格式化 ctrl+alt+f
以GBK编码打开文件（仅打开，非保存） ctrl+shift+c

粘贴到右边的用户设置框 

[
    {
        "keys": ["ctrl+alt+["],
        "command": "css_format",
        "args": {
            "action": "expand"
        }
    },
    {
        "keys": ["ctrl+alt+]"],
        "command": "css_format",
        "args": {
            "action": "compress"
        }
    }
]
