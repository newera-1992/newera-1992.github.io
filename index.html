注：如果遇到安装问题，一般是 GFW问题，需要自定义一个 channel_v3，搜索一下。

快捷键配置

路径：Perfercense > Key Bindings

粘贴到右边的用户设置框

[
    {
        "keys": ["ctrl+alt+["],
        "command": "css_format",
        "args": {
            "action": "expand"
        }
    },
    {
        "keys": ["ctrl+alt+]"],
        "command": "css_format",
        "args": {
            "action": "compress"
        }
    }
]

备份打包
